# 画面一覧（Screen List）

## ユーザー種別
- **C**: Consumer（一般ユーザー：デザイン作成・AR試着・予約）
- **S**: Salon（サロン・ネイリスト：入札・予約管理）
- **A**: Admin（運営）

---

## 画面遷移フロー（主要）

```
[ホーム(C-10)]
  ├─ デザイン詳細(C-12) → AR試着(C-30) → AR結果(C-31) → 予約リクエスト作成(C-40)
  │                                                              ↓
  │                                                        入札一覧(C-41) → 予約確定(C-43) → 完了(C-44)
  │
  └─ エディター(C-20) → パーツ選択(C-21) → 質感調整(C-22) → 保存/IP登録(C-23) → 結果(C-24)
                                                                                       ↓
                                                                                  AR試着(C-30)へ

[サロン側]
  ダッシュボード(S-03) → リクエスト詳細(S-04) → 入札(S-05)
  予約管理(S-06) → 詳細(S-07) → 施術完了 → ユーザーがレビュー(C-52)
```
